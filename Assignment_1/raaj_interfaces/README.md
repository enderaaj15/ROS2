In this interface package, a Car.msg file and BuyCar.srv files are created

The Car.msg is used in the publisher subscriber package (raajpubsub_py)
-using Car.msg allows the asking and answering of 'how many cars do the person have'.

The BuyCar.srv is used in the service client package (raajsrvcli_py)
-using BuyCar.srv allows a car buyer to buy a BMW with a specific model within his/her budget.
